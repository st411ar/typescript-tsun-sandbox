# typescript-tsun-sandbox
Typescript simple sandbox uses tsun utility

## requirements
- npm (nodejs package manager)

## setup
- general
```
npm install -g typescript
npm install -g tsun
```
- windows
```
setup
```

## start tsun
```
tsun
```

## stop tsun
```
ctrl+c
```

## run built-in-types-sandbox.ts
```
:clear
:load built-in-types-sandbox.ts
```
